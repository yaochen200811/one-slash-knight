This game need to install java to run.
Download at: https://www.java.com/en/

press up, down, left and right arrow key to move.
enter is confirm, backspace is cancel.

If you stay in the red area after move, you will lose
one shield. When you run out of shield, next damage will
kill you.

In infinity mode, every 5 points can add a new shield, and every 
20 points can unlock a new character when you died.

You can check enemies' attack range in option -> help

You can change your character in option -> choose character
